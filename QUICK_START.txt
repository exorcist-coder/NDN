🚀 QUICK START - GET YOUR APP RUNNING IN 5 MINUTES
===================================================

STEP 1: GET YOUR API KEYS (3 minutes)
─────────────────────────────────────

A) NewsAPI Key:
   1. Go to https://newsapi.org/register
   2. Sign up (use your email)
   3. Click email verification link
   4. Copy your API key
   5. Save it somewhere

B) Gemini API Key:
   1. Go to https://aistudio.google.com/apikey
   2. Sign in with Google account
   3. Click "Get API Key"
   4. Click "Create new API key in new project"
   5. Copy the key


STEP 2: SET UP YOUR ENVIRONMENT (2 minutes)
──────────────────────────────────────────

1. Create .env file in the news-analyzer folder
2. Add this content:

NEWSAPI_KEY=paste_your_newsapi_key_here
GEMINI_KEY=paste_your_gemini_key_here


STEP 3: INSTALL & RUN (2 minutes)
──────────────────────────────────

Open Terminal/Command Prompt in the news-analyzer folder:

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run news_analyzer.py


DONE! 🎉
A browser window opens automatically with your app!
Type something like "Tesla", "Bitcoin", or "AI News"
Hit the button and watch the magic happen!

TROUBLESHOOTING
───────────────

"API key error":
→ Make sure .env file is in the same folder as news_analyzer.py
→ Check you copied keys correctly (no extra spaces)

"streamlit command not found":
→ Run: pip install streamlit

"Connection error":
→ Check your internet connection
→ Make sure API keys have quota left
