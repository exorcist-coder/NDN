[TECHNICAL EXPLANATION - 2 minutes]
───────────────────────────────────

"Under the hood, here's what's happening:

1. STREAMLIT (Frontend)
   - Beautiful web interface, zero HTML coding needed
   - Loads in browser instantly
   - Responsive design works on phones too

2. NEWSAPI (Data Source)
   - Aggregates articles from 10,000+ publishers
   - Returns raw data (title, content, source, image, date)
   - REST API - simple HTTP requests

3. GOOGLE GEMINI (Summarization AI)
   - State-of-the-art language model
   - Understands context and meaning
   - Generates human-like summaries in seconds

4. TEXTBLOB (Sentiment Analysis)
   - Natural Language Processing library
   - Analyzes emotional tone of text
   - Assigns polarity score (-1 to +1)

5. CUSTOM CREDIBILITY ENGINE
   - Scores sources based on reputation database
   - Penalizes sensationalist language patterns
   - Combines both methods for final score (0-100%)"

[Q&A PREP - Common Questions & Answers]
──────────────────────────────────────

Q: "How do you know if the credibility scores are correct?"
A: "Great question! Our database is based on established journalism rankings 
   (BBC, Reuters, AP are top-rated). We also check for sensationalist language. 
   For absolute verification, users should cross-check with multiple sources. 
   This is a tool to help prioritize which sources to trust, not a replacement 
   for critical thinking."

Q: "What if someone disagrees with a credibility score?"
A: "The scoring is transparent - it's based on source reputation + language analysis. 
   Users can see why an article got a certain score and decide for themselves. 
   Our AI isn't claiming to be 100% accurate, just helpful."

Q: "Isn't this replacing human fact-checkers too much?"
A: "No, this is a first-pass filter. Professional journalists still need to verify facts. 
   But for regular people, this helps them avoid sharing obviously FALSE sources. 
   It's like spell-check vs. professional editor."

Q: "How did you build this so fast?"
A: "1. We used existing APIs (NewsAPI, Gemini, TextBlob)
   2. Streamlit made the UI super fast (no HTML needed)
   3. We focused on core features, not nice-to-haves
   4. Good planning beforehand"

Q: "Can this be commercialized?"
A: "Potentially! NewsAPI has commercial plans, and Gemini has enterprise pricing. 
   Journalists, newsrooms, or fact-checkers could use this. We'd need to improve accuracy 
   and add more features though."

Q: "Will AI-generated articles fool this system?"
A: "Good question! AI articles would likely score high on credibility (good grammar, 
   less sensationalism) but be completely false. We'd need to add AI-detection or 
   cross-reference with multiple sources."
